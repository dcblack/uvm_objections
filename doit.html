NAME

doit - Script to run various performance runs

SYNOPSIS

./doit [OPTIONS] [VERSIONS]

DESCRIPTION

This script uses a specified simulator (default Mentor questa) to run UVM objections performance testing. Default is to raise and lower objections millions of times and report the overall CPU time.

OPTIONS

-C #

Number of times to repeat inner loop

-D #

Number of agents to instantiate

-h

Display this help.

-L #

Number of levels of hierarchy above drivers

-P 0|1

Propagate objections or not (affects 1.2 and above only)

-q

Quiet mode to suppress unnecessary output.

-S questa|ius|vcs

Specify simulator to use.

-p #

Seconds to delay between simulations (minimum 5)

-v

Verbose mode to expand explanatory output.

VERSIONS

Supported versions are:

  1. 1b
  2. 1c
  3. 1d
  4. 2
  5. 2r
